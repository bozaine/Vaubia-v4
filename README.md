# Vaubia (clean bundle)

Projet React + Vite prêt pour Vercel.

## Démarrer en local

```bash
npm i
npm run dev
```

## Déployer sur Vercel

- Importez ce repo sur GitHub
- Vercel → New Project → Import
- Framework: Vite  | Build: `npm run build` | Output: `dist/`
- Ajoutez vos domaines (vaubia.com, www.vaubia.com)
```
